## About The Project

Avielle Institute is a single-page application designed as the main entry point for an online learning service. The platform is aimed at students preparing for standardized tests like the SAT, TOEFL, and IELTS, offering a variety of courses to help them achieve their academic goals.

The project emphasizes a great user experience with smooth animations, a responsive layout that works on all devices, and interactive components to engage users.

## Built With

This project is built with a modern frontend stack:

*   **React**: A JavaScript library for building user interfaces.
*   **Vite**: A next-generation frontend tooling that provides a faster and leaner development experience.
*   **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
*   **Shadcn/ui**: Re-usable components built using Radix UI and Tailwind CSS.
*   **Animate On Scroll (AOS)**: For subtle and effective scroll animations.
*   **Font Awesome & Lucide React**: For a rich set of icons.

## Features

-   **Fully Responsive Design**: Adapts seamlessly to desktops, tablets, and mobile devices.
-   **Dynamic Counters**: Animated counters for lessons and student numbers to showcase platform growth.
-   **Interactive Course Cards**: A grid of available courses with hover effects and enrollment CTAs.
-   **Testimonial Slider**: An auto-playing carousel of student testimonials to build social proof.
-   **Smooth Scrolling**: "Back to Top" button and smooth navigation to different sections.
-   **Engaging Animations**: On-scroll animations for various sections to enhance user engagement.
-   **Clear Call-to-Actions**: Strategically placed buttons for user conversion, such as "Get Started", "Enroll Now", and "UPGRADE".
-   **Newsletter Subscription**: A functional footer section for users to subscribe to a newsletter.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine.
*   npm
    ```sh
    npm install npm@latest -g
    ```

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username/my-elearning-app.git
    ```
2.  Navigate to the project directory
    ```sh
    cd my-elearning-app
    ```
3.  Install NPM packages
    ```sh
    npm install
    ```

## Usage

To run the app in development mode:

```sh
npm run dev
```
This will open the app in your default browser at `http://localhost:5173`.

To build the app for production:

```sh
npm run build
```
This command bundles the app into static files for production in the `dist` folder.
